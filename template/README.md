# Skill Template

Use this as a starting point for new skills.

## Quick Setup

```bash
# Copy this directory
cp -r template my-skill
cd my-skill

# Replace placeholders:
# - SKILL_NAME
# - SHORT_DESCRIPTION
# - AUTHOR
```

## Files to Edit

1. `README.md` — User documentation
2. `SKILL.md` — OpenClaw manifest
3. `scripts/skill.sh` — Implementation
4. `LICENSE` — Your name

## Placeholder Reference

- `SKILL_NAME` — kebab-case name (e.g., "weather-check")
- `SHORT_DESCRIPTION` — One sentence
- `LONG_DESCRIPTION` — Paragraph explaining purpose
- `AUTHOR` — Your name or handle
- `VERSION` — Start with "0.1.0"
